# Portfolio site by maximobesteiro

## Summary

This is my portfolio site. I built it with [Next.js](https://nextjs.org), React, TypeScript, TailwindCSS and a lot of love ❤️.

## Running locally

Just clone and run:

```bash
pnpm install
pnpm dev
```

Then, open [http://localhost:3000](http://localhost:3000).
